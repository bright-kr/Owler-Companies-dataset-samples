# Owler Companies 데이터셋 샘플

이 **Owler Companies Dataset Sample**은 **1,030개 레코드**를 포함하며 **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

|------------|-------------|
| `companyId` | 각 회사를 위한 고유 식별자입니다. |
| `ownership` | 회사가 비상장인지 상장인지 여부를 지정합니다. |
| `industrySectors` | 회사가 운영하는 산업 섹터를 나열합니다. |
| `revenue` | 회사의 연간 매출(USD)입니다. |
| `founded` | 회사가 설립된 연도입니다. |
| `companyName` | 회사의 공식 명칭입니다. |
| `country` | 회사가 기반을 두고 있는 국가입니다. |
| `employeeCount` | 회사에서 근무하는 총 직원 수입니다. |
| `totalAcquisitions` | 회사가 수행한 인수의 누적 횟수입니다. |
| `zipcode` | 회사 주요 소재지의 우편번호입니다. |
| `website` | 회사 공식 웹사이트 URL입니다. |
| `domainName` | 회사 웹사이트의 도메인 이름입니다. |
| `phoneNumber` | 회사의 연락처 전화번호입니다. |

| `followers` | Owler에서 해당 회사를 팔로우하는 사람 수입니다. |

| `teamName` | 회사 내 팀 또는 부서의 이름입니다. |

| `city` | 회사 본사가 위치한 도시입니다. |

## Use Cases

### 1. Market Research
데이터셋을 활용하여 다양한 산업 섹터 전반의 산업 트렌드와 여러 회사의 시장 포지셔닝을 분석할 수 있습니다.

### 2. Competitor Analysis
핵심 경쟁사를 식별하고 비즈니스 지표를 평가하여 경쟁 환경에서 효과적으로 전략을 수립할 수 있습니다.

### 3. Investment Insights
회사 재무 및 투자 라운드에 대한 인사이트를 확보하여 정보에 기반한 투자 의사결정을 내릴 수 있습니다.
